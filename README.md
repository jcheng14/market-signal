# Prediction Market Intelligence Brief

An open-source tool that aggregates prediction market data from **Polymarket**, **Kalshi**, and **Manifold** into actionable intelligence briefs — covering AI/tech, geopolitics/trade, and crypto.

> Data is opinion. These are probabilities generated by thousands of traders betting real money — more accurate than polls, faster than experts, earlier than news.

## What This Does

```
3 prediction markets  -->  Automated data collection  -->  Curated intelligence brief
(Polymarket, Kalshi,       Cross-platform comparison       Markdown / HTML / PDF
 Manifold)                 Signal filtering & ranking       Ready for newsletter
```

**For each prediction market poll, you get:**
- Current probability + 7-day change across platforms
- Cross-platform price differences (a signal in itself)
- Related news context
- Actionable interpretation for entrepreneurs, traders, and decision-makers

## Quick Start

### Prerequisites

- [Claude Code](https://claude.ai/claude-code) installed
- API access (Polymarket and Manifold are free, Kalshi requires registration)

### Generate a Brief

```bash
# Clone the repo
git clone https://github.com/jcheng14/market-signal.git
cd market-signal

# Run with Claude Code
claude --skill ng-brief
```

Or manually trigger the skill:

```bash
claude "Generate a prediction market intelligence brief for this week"
```

The skill will:
1. Query Polymarket Gamma API, Manifold Markets API, and Kalshi
2. Filter by volume, probability change, and settlement window
3. Cross-reference the same events across platforms
4. Generate a formatted Markdown brief with analysis

## Data Sources

| Platform | Type | Auth | Rate Limit |
|----------|------|------|-----------|
| **Polymarket** | Real money (USDC) | None (public API) | ~100 req/min |
| **Manifold** | Play money (Mana) | None (public API) | 500 req/min |
| **Kalshi** | Real money (CFTC-regulated) | API key required | Tiered |

### API Endpoints

```
Polymarket:  https://gamma-api.polymarket.com/markets?active=true
Manifold:    https://api.manifold.markets/v0/search-markets?term={query}
Kalshi:      https://api.elections.kalshi.com/trade-api/v2/markets
```

## Filtering Logic

Polls are ranked by a weighted score:

| Factor | Weight | Description |
|--------|--------|-------------|
| Probability change (7d) | 40% | Larger swings = more newsworthy |
| Trading volume | 30% | Higher liquidity = more reliable signal |
| Time to settlement | 20% | 7-90 day window preferred |
| Cross-platform coverage | 10% | Same event on multiple platforms = bonus |

## Configuration

See [docs/config.yaml](docs/config.yaml) for full configuration options:

```yaml
filtering:
  min_probability_change_7d: 5       # minimum 7-day change (percentage points)
  highlight_threshold_7d: 10         # mark as major move
  settlement_window_days: [7, 90]    # only select markets in this range

output:
  include_dashboard: true            # summary table
  include_content_ideas: true        # social media angles
  include_cross_platform_diff: true  # highlight pricing gaps
```
